# RoomViz Frontend

A high-performance React application for 3D room visualization. Built on Vite for a lightning-fast developer experience.

## 🚀 Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

3.  **Open in browser**:
    Vite will give you a local URL (e.g., `http://localhost:5173`).

---

## 🏗 Tech Stack

- **React 19**: Core UI framework.
- **Three.js & @react-three/fiber**: 3D rendering and physics.
- **Zustand**: Lightweight global state management.
- **Framer Motion**: Smooth UI transitions.
- **Vite**: Modern frontend build tool.

---

## 📂 Highlights

- **Dynamic 3D Rooms**: Realistic lighting and materials.
- **Responsive UI**: Works seamlessly on different screen sizes.
- **Backend Integration**: Connects to the Node.js API at `http://localhost:5000/api`.
